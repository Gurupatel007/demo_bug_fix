# Demo Bug Fix Repository

Welcome! This repository contains intentional bugs for practice.

## How to Contribute

1. **Find a bug** by reviewing the code
2. **Open an issue** describing the bug
3. **Fork this repository**
4. **Create a branch** for your fix
5. **Fix the bug** in your branch
6. **Open a pull request** back to this repository
